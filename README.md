# تطبيق أثير التجريبي (Atheer SDK Demo App)

تطبيق أثير التجريبي هو تطبيق أندرويد متكامل يهدف إلى استعراض وإثبات قدرات مكتبة **Atheer SDK**. يقدم التطبيق بيئة محاكاة كاملة لعمليات الدفع اللاتلامسي عبر تقنية (NFC)، حيث يحتوي على واجهتين رئيسيتين:

1.  **واجهة التاجر (SoftPOS):** لاستقبال المدفوعات مباشرة عبر الهاتف.
2.  **واجهة العميل (HCE):** لإجراء المدفوعات اللاتلامسية باستخدام البطاقات المحفوظة.

## ✨ المميزات الرئيسية

*   **نقاط البيع البرمجية (SoftPOS):** تمكين هواتف التجار من العمل كجهاز نقاط بيع (POS) لاستقبال المدفوعات عبر الـ NFC.
*   **محاكاة البطاقات (HCE):** رقمنة البطاقات البنكية للعملاء والدفع من خلال تمرير الهاتف على جهاز التاجر.
*   **المصادقة الحيوية (Biometric):** تأمين عمليات الدفع للعملاء باستخدام البصمة أو التعرف على الوجه قبل تنفيذ العملية.
*   **دعم وضع الأوفلاين (Offline):** معالجة البيانات وتخزينها محلياً لضمان استمرارية العمل (عبر Room Database).

---

## 🛠 التقنيات المستخدمة (Tech Stack)

*   **لغة البرمجة:** Kotlin (100%)
*   **نظام التشغيل:** Android (Min SDK: 26, Target SDK: 34)
*   **المعمارية:** MVVM (Model-View-ViewModel)
*   **الشبكات:** Retrofit2 & OkHttp3
*   **المهام غير المتزامنة:** Kotlin Coroutines & Flows
*   **قاعدة البيانات المحلية:** Room Database
*   **المكتبة الأساسية:** `atheer-sdk-release.aar`

## ⚙️ المتطلبات الأساسية للتشغيل

قبل البدء، تأكد من توفر المتطلبات التالية:

2.  هاتف أندرويد حقيقي يدعم تقنية **NFC** (الـ Emulator لن يفيد في تجربة عمليات الدفع اللاتلامسي).
3.  اتصال بالإنترنت (للتسجيل وتسجيل الدخول الأولي).

## 🚀 الإعداد والتشغيل (Installation & Setup)

لتشغيل المشروع على جهازك المحلي، اتبع الخطوات التالية:

1.  **استنساخ المستودع:**
    ```bash
    git clone https://github.com/your-username/atheer_app.git
    cd atheer_app
    ```

2.  **إعداد مكتبة أثير (Atheer SDK):**
    *   التطبيق التجريبي يعتمد حالياً على نسخة محلية من المكتبة بصيغة `.aar`.
    *   تأكد من وجود الملف `atheer-sdk-release.aar` داخل المسار التالي: `demo-app/libs/atheer-sdk-release.aar`.
    *   *ملاحظة:* لا حاجة لإعداد `GITHUB_TOKEN` طالما أنك تستخدم الملف المحلي المرفق.

3.  **تشغيل التطبيق:**
    *   افتح المشروع باستخدام Android Studio.
    *   قم بعمل **Sync Project with Gradle Files**.
    *   قم بتوصيل هاتفك (الداعم لـ NFC) واضغط على **Run**.

## 🔒 الصلاحيات المطلوبة (Permissions)

يطلب التطبيق الصلاحيات التالية لضمان عمل كافة الميزات:
*   `NFC`: للاتصال اللاتلامسي بين هاتف العميل وهاتف التاجر.
*   `INTERNET`: للاتصال بخوادم أثير والمزامنة.
*   `USE_BIOMETRIC`: للتحقق من هوية العميل قبل الدفع.
*   `ACCESS_NETWORK_STATE`: للتحقق من حالة الاتصال.
*   `VIBRATE`: لإعطاء رد فعل لمسي (Haptic Feedback) عند نجاح أو فشل عملية الدفع.
