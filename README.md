# Atheer Demo App - تطبيق تجريبي لنظام أثير (SoftPOS & NFC)

تطبيق أندرويد تجريبي (Demo) مصمم لاستعراض قدرات **Atheer SDK**، وهي مكتبة متخصصة في حلول الدفع الرقمي عبر تقنية NFC، مما يحول الهواتف الذكية إلى نقاط بيع (SoftPOS) أو يحاكي عمل البطاقات البنكية (HCE).

## 🚀 المميزات الرئيسية

*   **تكامل Atheer SDK:** عرض كامل لعمليات الدفع والتحقق.
*   **نقطة بيع برمجية (SoftPOS):** استقبال المدفوعات مباشرة عبر تقنية NFC.
*   **محاكاة البطاقة (HCE):** تمكين العميل من استخدام هاتفه كبطاقة دفع.
*   **نظام أدوار (Roles):** واجهات مخصصة لكل من "التاجر" و "العميل".
*   **الأمان:** دعم البصمة (Biometric Prompt) وتخزين البيانات المشفر (Security Crypto).
*   **واجهة مستخدم عصرية:** استخدام Material Design 3 ورسوم متحركة عبر Lottie.

## 🏗 بنية المشروع (Project Structure)

يتبع المشروع هيكلية واضحة تفصل بين المنطق وواجهة المستخدم:

*   **`:demo-app`**: الوحدة الرئيسية للتطبيق.
    *   `ui/`: يحتوي على الأنشطة (Activities) والواجهات مقسمة حسب الوظيفة:
        *   `login/`: التسجيل وتسجيل الدخول.
        *   `dashboard/`: لوحة التحكم لاختيار وضع التشغيل.
        *   `pos/`: واجهة نقطة البيع للتاجر.
        *   `customer/`: واجهة العميل لمحاكاة البطاقة.
        *   `payment/` & `result/`: معالجة نتائج العمليات.
    *   `data/`: التعامل مع البيانات (نماذج البيانات وقواعد البيانات).
    *   `libs/`: يحتوي على ملف `atheer-sdk-release.aar` (المكتبة الأساسية).

## 🛠 المتطلبات التقنية

*   **لغة البرمجة:** Kotlin
*   **أقل إصدار أندرويد (Min SDK):** 26 (Android 8.0)
*   **إصدار الهدف (Target SDK):** 34 (Android 14)
*   **أدوات البناء:** Gradle 8.x مع Kotlin DSL (`.kts`)
*   **المكتبات المستخدمة:**
    *   `ViewBinding`: لربط الواجهات.
    *   `Coroutines`: للعمليات الخلفية.
    *   `Retrofit` & `Gson`: للاتصال بخدمات الويب.
    *   `Room`: لإدارة قواعد البيانات المحلية.
    *   `Biometric`: للتحقق من الهوية.

## ⚙️ الإعداد والتشغيل

1.  **استنساخ المشروع:**
    ```bash
    git clone https://github.com/atheer/atheer-demo-app.git
    ```
2.  **إعداد مفاتيح الوصول:**
    تأكد من توفر صلاحية الوصول لمستودع Atheer على GitHub في ملف `settings.gradle.kts` عبر متغيرات البيئة `GITHUB_ACTOR` و `GITHUB_TOKEN`.
3.  **تهيئة الـ SDK:** يتم تهيئة المكتبة تلقائياً في فئة `AtheerDemoApp.kt`:
    ```kotlin
    AtheerSdk.init(
        context = this,
        merchantId = "777000000",
        apiBaseUrl = "https://sea-lion-app-jnyrn.ondigitalocean.app"
    )
    ```
4.  **التشغيل:** افتح المشروع في Android Studio وقم بعمل `Sync Project with Gradle Files` ثم اضغط على `Run`.

## 🔒 الصلاحيات المطلوبة

يتطلب التطبيق الصلاحيات التالية للعمل بشكل صحيح:
*   `NFC`: للتواصل مع البطاقات أو الهواتف الأخرى.
*   `INTERNET`: للتحقق من المعاملات مع الخادم.
*   `USE_BIOMETRIC`: لتأمين الدخول والدفع.
